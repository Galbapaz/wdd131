Site Plan

This page explains the purpose, audience, structure, and design of the website.

Site Name

English Learning Hub

This website provides resources, study tips, and interactive tools to help students improve their English skills.

Purpose

The purpose of this website is to help English learners practice vocabulary, listening, speaking, and grammar through simple and useful activities.

Target Audience

This website is designed for beginner and intermediate English learners who want practical ways to improve their language skills.

User Needs

Users need a website that is simple, visually clear, mobile-friendly, and useful for daily English practice.

Site Structure

  • Home: Introduction and featured study advice.
  • Resources: Learning resources with filters and favorites.
  • Study Tips: Advice and motivation for learning English.
  • Contact: Form for user interaction.
  • References: Sources used in the website.
  • Site Plan: Website planning information.

Color Scheme

Primary: #1d3557

Secondary: #457b9d

Accent: #f1faee

These colors create a clean and professional educational look.

Typography

Font: Roboto

This font is clean, modern, and easy to read on all devices.

Dynamic Features

  • Responsive navigation menu
  • Random daily tip (Home)
  • Motivation button (Study Tips)
  • Resource filtering (Resources)
  • Favorites saved with localStorage
  • Interactive form response

Wireframe Description

The website uses a consistent layout with a header, navigation menu, content sections, and footer. The homepage includes a hero section, and internal pages use a card-based design.
