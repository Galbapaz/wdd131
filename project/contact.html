Site Plan

This page explains the purpose, audience, structure, and design of the website.

Site Name

English Learning Hub

This name was chosen because the website provides useful English learning resources, study tips, and interactive tools for students.

Purpose

The purpose of this website is to help students improve their English skills through study resources, learning advice, and interactive features.

Target Audience

This site is designed for beginner and intermediate English learners who want practical and simple ways to improve vocabulary, listening, speaking, and grammar.

User Needs

Users need a website that is easy to navigate, visually clear, mobile friendly, and useful for daily English practice.

Site Structure

  • Home: introduction and featured study advice.
  • Resources: categorized learning resources with filtering and favorites.
  • Study Tips: useful advice and motivation for learning English.
  • Contact: student form for feedback and interaction.
  • References: list of sources used in the project.
  • Site Plan: planning information for the website.

Color Scheme

Primary Color: #1d3557

Secondary Color: #457b9d

Accent Color: #f1faee

These colors create a clean, calm, and academic look that matches the educational purpose of the website.

Typography

Font: Roboto

Roboto was selected because it is clean, modern, easy to read, and appropriate for educational content on both desktop and mobile devices.

Dynamic Features

  • Responsive navigation menu
  • Random daily tip on the home page
  • Motivation button on the Study Tips page
  • Category filtering on the Resources page
  • Favorite resources saved with local storage
  • Interactive contact form response

Wireframe Description

The homepage uses a hero section with text and an image, followed by content cards. The internal pages keep a consistent header, navigation, footer, and card-based layout. The design is responsive for small and large screens.
